My Portfolio Website

Designing, coding, and customizing a personal site with Bootstrap

Information Architecture

I designed the global navigation to include six main links: Home, About, Portfolio, Resume, Blog, and Contact. These appear on every page via a fixed-top Bootstrap navbar. The Portfolio page uses modals for each project, while the Resume page features clearly labeled tables. Associative navigation—such as “Explore Now” buttons—directs users through the site intuitively, and the footer provides external links for email and social platforms. This structural and utility navigation ensures consistency and easy access for all users.

Design Research

My research process involved analyzing portfolio templates from Start Bootstrap and BootstrapMade, as well as browsing Behance and Webflow for aesthetic inspiration. I created a mood board to establish a neon color palette that reflects my personal brand. I also used the Bootstrap documentation to ensure responsive components were implemented correctly, including grids, modals, cards, and tables. These insights guided both my UX decisions and technical build process.

Design Adjustments

I iterated significantly on the site's layout. The Portfolio page evolved from text-heavy sections into a thumbnail grid with modals to improve clarity. The About page now uses a timeline for milestones. The Home page highlights skills with animated progress bars. Personal touches—like a handwritten logo and embedded video on the Contact page—add warmth and uniqueness. Animations and a disappearing navbar enhance user experience without sacrificing clarity.

Bootstrap Implementation & Responsive Design

Throughout the site, I relied on Bootstrap's grid system, navbar, modals, tables, accordions, and progress bars. Every component was customized with CSS to match a neon aesthetic, including gradient hover effects, handwritten fonts, and dark backgrounds. Cards and buttons include subtle animations, and accessibility was supported with high-contrast color choices, alt text, and keyboard navigation compatibility. Using Bootstrap’s breakpoints and media queries, the site adjusts cleanly across desktops, tablets, and mobile devices. All visual assets were compressed and resized for fast load times and accessibility across screen types.

Usability & Accessibility

Following WCAG 2.0 standards, I included ARIA labels, alt attributes, and keyboard-friendly navigation. Font sizes scale with rem units, and all interactions provide visual feedback via hover or focus states. I conducted informal testing with peers and family to improve clarity, contrast, and navigation speed. These measures ensure that my site remains inclusive for users across a range of abilities.
